ESA — Wireframes

Low-fidelity wireframes for ESA — layout, navigation, and key actions per screen before full UI styling. Screenshot each numbered section for README evidence.

Public marketing page for visitors and new schools. Hero explains the platform; three feature cards highlight multi-tenant isolation, teacher sign-off, and Stripe Connect fees. Top nav: Home, Subscribe, Log in, Register. Primary CTAs: Get Started and Learn More.

3. Login
Shared login for Super Admin, School Admin, Teacher, Student, and Parent. Email and password fields, forgot-password link, Sign In button, and link to registration. After login, user is routed to the dashboard matching their role.

Account creation with first/last name, email, password, confirm password, and role dropdown (Student, Parent, Teacher, etc.). School linkage will be required for non–super-admin roles at the API layer. Links back to login for existing users.
