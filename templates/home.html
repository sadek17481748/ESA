ESA Education & Schooling Applications

ESA platform

Multi-tenant school management for Islamic schools — Hifz, Alimiyah, attendance, homework, and parent payments.

  • Log in — parents and staff (session auth)
  • Parent payments — fees and Stripe checkout
  • Admin — Django admin
  • API token — JWT for REST clients

Demo users: run python manage.py seed_rbac_users on the server, then log in as schooladmin, teacher_demo, parent_demo, or student_demo.
